# Zosima Quartet

Single-page website for GitHub Pages.

## Kde upravovat obsah

- České texty: `content/cs.json`
- Anglické texty: `content/en.json`
- E-mail, Facebook a YouTube videa: `content/settings.json`
- Logo: `assets/images/zosima-logo.png`

Po úpravě souboru na GitHubu klikněte na **Commit changes**. Web se obvykle aktualizuje během 1–2 minut.

## GitHub Pages

Settings → Pages → Deploy from branch → Branch: `main` → Folder: `/root` → Save.
